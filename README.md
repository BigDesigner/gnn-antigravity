# GNN-antigravity WordPress Theme

A highly minimalist, experimental "Canvas" theme mirroring the actual antigravity.google design language. Built with zero 3rd-party dependencies, prioritizing 100% Native WP API integrations and robust performance.

## Features
- **Glassmorphism Design:** Deep scroll-based visual cues, sticky headers, and smooth transitions.
- **Native Hero Slider:** Zero-dependency, GSAP-powered, and touch-enabled image/content slider fully manageable via the WordPress Customizer.
- **Static Hero Image:** Full-width static background image option for the front page when the slider is disabled, with adjustable overlay opacity.
- **Magnetic Custom Cursor:** GSAP-driven custom cursor overlay with high z-index stacking.
- **Elementor Support:** Ships with `Full Width` and `Canvas` templates adhering strictly to Elementor standards.
- **Advanced Native SEO:** Meta tags, Open Graph, Twitter Cards, and JSON-LD Schema (with auto-disable if SEO plugins are detected).
- **Bot Protection:** Native Cloudflare Turnstile and Google reCAPTCHA v3 fallback integrations for WordPress forms.
- **Analytics:** Native GA4 integration.
- **Complete Template Hierarchy:** Fully styled `single`, `archive`, `search`, `404`, and `comments` pages.
- **GitHub Auto-Updater:** Self-hosted update system that checks GitHub Releases for new versions and allows one-click updates from the WP Admin dashboard. Controllable via Customizer toggle and manual check button.

## Installation
1. Download the theme repository as a zip or clone it into `wp-content/themes/gnn-antigravity`.
2. Activate via **Appearance > Themes**.
3. Configure settings via **Appearance > Customize**.
4. Check for updates via **Appearance > Theme Updates**.
