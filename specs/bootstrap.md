# Project Bootstrap

## 1. Environment Setup
- **PHP:** 7.4+
- **WordPress:** Latest stable version.
- **Local Development:** Recommended using LocalWP or Docker (wp-env).

## 2. Dependency Map
- **Core:** WordPress
- **Theme Logic:** `functions.php`, `inc/` directory.
- **Styling:** `style.css`, `assets/css/`.
- **Scripts:** `assets/js/`.

## 3. Installation Steps
1. Clone the repository into `wp-content/themes/`.
2. Run `npm install` (if build tools are added).
3. Activate the theme via WordPress Dashboard.
